# 四柱推命な日常がきこえる - Web Site

このプロジェクトは、自然派四柱推命ポッドキャスト「四柱推命な日常がきこえる」の公式ウェブサイトです。
Viteを使用して構築されており、モダンなデザインとスムーズなアニメーションが特徴です。

## 技術スタック
- **Vite**: ビルドツール & 開発サーバー
- **Tailwind CSS**: レイアウト & スタイリング
- **Vanilla CSS**: カスタムデザイン & トークン
- **Vanilla JS**: スクロールアニメーション (Reveal.js スタイル)

## 起動方法
1. ターミナルで以下のコマンドを実行します：
   ```bash
   npm run dev
   ```
2. 表示されたURL（通常は `http://localhost:5173`）をブラウザで開いてください。

## デザインのポイント
- **和洋折衷なタイポグラフィ**: 落ち着いた「しっぽり明朝」とモダンな「Inter/Noto Sans JP」を組み合わせています。
- **プレミアムな質感**: グラスモーフィズム（透過ぼかし）のヘッダーや、洗練されたカラーパレットを採用。
- **ダイナミックな演出**: スクロールに合わせてコンテンツが浮き上がるアニメーションや、画像のホバーエフェクトを実装しています。
